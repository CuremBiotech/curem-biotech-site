The Team

We are a group of students from IISER Pune, taking up the work from iGEM 2020 forward to a bio startup.

What we do

We are working on developing a fast and efficient diagnosis method using Machine Learning methods, which should be able to cut costs and diagnosis times dramatically.

Hardware

Building small and efficient imaging hardware for sampling

Data Collection

Building a database of actual samples for improving diagnostic systems

Machine Learning

Applying ML techniques to identify and classify samples

Software

Building software tools accompanying the imaging hardware and ML models
